Robot & Maze
Data Structure & Algorithm
I was building a maze-navigating robot. The robot starts randomly. Each step, the robot may travel up, down, left, or right. If the new position is unattainable, such a wall, the robot receives a "false" signal and stays put. If not, a "true" signal will move it. The robot is blind in the dark labyrinth. Develop algorithms and data structures to enable the robot reach the Exit. The robot gets a "win" signal at the Exit gate.

As more people book cars online, automobile rental firms have grown in popularity. However, our car rental website provides a driver with the rental vehicle.
